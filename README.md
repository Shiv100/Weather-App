# Weather App

A simple and modern weather application that shows current weather information for any city worldwide.

## Features

- Search for any city worldwide
- Display current temperature in Celsius
- Show weather description with icon
- Display wind speed and humidity
- Responsive design that works on mobile and desktop
- Modern glassmorphism UI design

## How to Use

1. Open `index.html` in your web browser
2. Enter a city name in the search box
3. Click the search button or press Enter
4. View the weather information for the selected city

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- OpenWeatherMap API

## API Key

This app uses the OpenWeatherMap API. The API key is included in the code, but if you need to use your own:

1. Sign up at [OpenWeatherMap](https://openweathermap.org/)
2. Get your API key from your account dashboard
3. Replace the API key in `script.js`
